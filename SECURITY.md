# Security Policy

## Supported Versions

Only the latest minor release of `adaptive_navbar` receives security fixes.

| Version  | Supported          |
| -------- | ------------------ |
| 1.0.x    | :white_check_mark: |
| < 1.0    | :x:                |

## Reporting a Vulnerability

`adaptive_navbar` is a UI widget package — it does not handle network, auth,
or user data — so the practical security surface is small. That said, if you
believe you've found a vulnerability (for example, an issue in a transitive
dependency exposed via this package), please **do not open a public issue**.

Report it privately via GitHub Security Advisories:

➡️ [Report a vulnerability](https://github.com/moulibheemaneti/adaptive_navbar/security/advisories/new)

You can expect:
- An acknowledgement within **7-14 days**.
- A status update within **30-45 days**.
- If accepted, a fix will land in the next patch release and you'll be credited in the release notes (unless you prefer to remain anonymous).
- If declined, you'll receive an explanation of why.
